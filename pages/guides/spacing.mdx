# Spacing

## Margin

|                        | Usage                                                          | Large | Medium | Small |
| ---------------------- | -------------------------------------------------------------- | ----- | ------ | ----- |
| Before content section | Spacing between a text link and an independent content section | 64px  | 56px   | 48px  |
| Display 3 to text      | Spacing between Display 3 to Body                              | 16px  | 16px   | 16px  |
| Display 4 or 5 to text | Spacing between Display 4 or 5 to Body or Body / Small         | 8px   | 8px    | 8px   |

## Padding

|         |                                                                                                  | Large | Medium | Small |
| ------- | ------------------------------------------------------------------------------------------------ | ----- | ------ | ----- |
| Default | Top and bottom padding                                                                           | 128px | 88px   | 64px  |
| Reduced | Top and bottom padding within a de-emphasized block or two blocks with the same color background | 88px  | 64px   | 48px  |
