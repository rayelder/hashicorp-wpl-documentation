# KeyBenefitsWithImage

## Purpose

To promote product or solution key benefits alongside imagery

## Rules

* Limit of 3 bullets

## Props

| Name | Type | Description |
|----|----|----|
| `appearance` | `string` |    |
| `showNumber` | `boolean` |    |
| `heading` | `string` |    |
| `description` | `string` |    |
| `whyHeading` | `string` |    |
| `why` | `string` |    |
| `bulletsHeading` | `string` |    |
| `bullets` | `string` |    |
| `showImage` | `boolean` | Show or hide the image. |

## States

* Not applicable

## UX writing guidelines

* Focus on writing keyword relevant display titles
* Target a readability grade level score of 8 to meet plain language guidelines

### Character limits

| Content | Minimum | Maximum |
|----|----|----|
| `heading` | 40 | 55 |
| `description` | 75 | 100 |
| `whyHeading` | 40 | 55 |
| `why` | 150 | 170 |
| `bulletsHeading` | 40 | 55 |
| `bullets` (each bullet) | 40 | 55 |
