## Figma props

| Name | Type | Description |
|----|----|----|
| `page` | `variant` | Display component on `page` backgrounds. **Options:** `light` or `strong` |
| `state` | `variant` | Toggle between `enabled` and `disabled` states. **Default:** `enabled` |
